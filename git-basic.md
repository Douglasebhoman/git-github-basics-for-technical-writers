# Introduction to Git & GitHub for Technical Writers

This document is designed to help technical writers understand the basics of Git and GitHub.  
It covers key concepts, workflows, and terminology to help writers create and manage documentation in a developer-friendly environment.

## Audience

This guide is intended for:
- Technical writers new to version control
- Documentation specialists working with developer teams
- Anyone interested in understanding GitHub workflows for documentation purposes

## What You Will Learn

- What Git is and why it matters
- What GitHub is and how it differs from Git
- Basic workflows for writing and maintaining documentation
- Common terminology every technical writer should know
